# CPF and CNPJ Skill

Provides endpoints for validating and formatting CPF and CNPJ document numbers. The tools use Python scripts designed for validation with unit tests.

# examples

## validate_cpf

```bash
python scripts/validate_cpf.py 12345678909
```

## format_cpf

```bash
python scripts/format_cpf.py 12345678909
```

## validate_cnpj

```bash
python scripts/validate_cnpj.py 12345678909
```

## format_cnpj

```bash
python scripts/format_cnpj.py 12345678909
```
